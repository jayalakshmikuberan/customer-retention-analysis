 Customer Retention Analysis

  Project Overview
This project analyzes customer behavior to identify churn patterns and predict customer retention.

 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab

 Key Insights
- Customers with high recency are more likely to churn
- Higher usage frequency leads to better retention
- Support calls may indicate dissatisfaction

 Model
Logistic Regression model used for churn prediction.

 Outcome
Provides insights to improve customer retention strategies.
